# 需求:常用 CV 顯示調整工具箱(平常收合)

> `/user` 階段產物:我只講「我想要什麼、為什麼」。怎麼實作交給 PO/architect。

## 1. 痛點

有些圖對比很低、太暗、太亮,或雜訊多,肉眼不容易一眼看出重點。我想要在看圖的當下能夠
臨時調整一下顯示效果,幫助我看得更清楚。

## 2. 我想要的

- 一個平常收合、要用才展開的工具箱,裡面放我做 CV 時常用的調整:
  - 亮度、對比
  - Gamma 調整
  - 直方圖均衡化(Histogram Equalization)
  - 反色(Invert)
  - 對比度極限拉伸(min-max stretch)
  - 二值化
  - Canny(邊緣偵測)
- 這些調整**只影響我看到的畫面**,不影響實際的偵測框判定,也不寫進匯出的檔案。
- 切換圖片或關掉工具箱後,恢復原本的顯示。

## 3. 我不在乎 / 可放棄

- 不需要把調整後的結果存檔或匯出。
- 不需要影響縮圖牆(縮圖維持原樣,這個工具箱只影響主圖)。
