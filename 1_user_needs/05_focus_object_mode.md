# 需求:Focus Object 模式——自動放大到最高信心物件

> `/user` 階段產物:我只講「我想要什麼、為什麼」。怎麼實作交給 PO/architect。

## 1. 痛點

我每天大部分時間在看同一件事:這張圖 YOLO 判斷得準不準。但現在每張圖預設都是 fit 到整張、
信心最高的框可能很小、埋在畫面某個角落,我還要自己滾輪縮放、找到它、看清楚。切下一張又要重來一次。

## 2. 我想要的

- 一個可以開關的「Focus Object」模式。開啟後,**每次換圖都自動放大到這張圖裡信心最高的那個物件**,
  讓我一眼就看到 YOLO 覺得最有把握的判斷是不是真的對。
- 如果我有調信心門檻或 Object 類別篩選,自動放大應該跟著「目前畫面上實際顯示的框」走,
  不要跳去一個我已經篩掉、根本沒顯示的框。

## 3. 我不在乎 / 可放棄

- 沒有偵測框的圖,正常顯示整張圖就好,不用勉強放大到哪裡。
- 不需要記住我手動縮放過的位置——這個模式的重點就是幫我自動跳到重點,不是取代我自己操作。
